初识WebDriver

  1. WebDriver原理：

    Selenium Webdriver是通过各种浏览器的驱动（web driver）来驱动浏览器，原理示意图如下：

  1. 在python中安装WebDriver：

完成python，pip工具的安装后，在cmd下输入“pip install selenium”，

完成安装后输入”python” 进入python 交互模式， 输入“import selenium” 没有报错即成功安装selenium。

  1. 安装浏览器驱动：

Firefox 浏览器的驱动： https://github.com/mozilla/geckodriver/releases

Google浏览器驱动： http://npm.taobao.org/mirrors/chromedriver/

选择适合自己操作系统的版本进行下载。

  1. 第一个webDriver 脚本：
    #-*-coding:UTF-8-*-
    fromseleniumimportwebdriver
    fromselenium.common.exceptionsimportNoSuchElementException
    importtime
    importselenium.webdriver.support.uiasui
    
    #通过executable_path参数指明Firefox驱动文件所在路径
    #路径查询：windows系统或者采用C:/Users/34954/Downloads/geckodriver
driver=webdriver.Firefox(executable_path="C:\\Users\\34954\\Downloads\\geckodriver")
    
    #若定位元素时间较长，可用10s内每隔500ms扫描1次页面变化，当出现指定元素后结束
    wait=ui.WebDriverWait(driver,10)
    #打开百度首页
    driver.get("https://baidu.com")
    
    try:
        wait.until(lambdadriver:driver.find_element_by_id("kw"))
    #清空搜索输入框默认内容
        driver.find_element_by_id("kw").clear()
    #在搜索输入框内输入“自动化测试”
        driver.find_element_by_id("kw").send_keys("自动化测试")
    #单击“搜索”按钮
        driver.find_element_by_id("su").click()
    #若定位不到元素，抛出异常
    exceptNoSuchElementExceptionasmsg:
        print("查找元素异常")
    
    #等待3秒
    time.sleep(3)
    #退出浏览器
    driver.quit()

4.1 问题及建议

问题 1. 报错提示NoSuchElementException: Unable to locate element

解决办法： 去网页相应的元素下查找相关的element_id， 如url为sogou.com时对应的查询id为“query”，对应的搜索button id为“stb”
